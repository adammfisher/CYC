CyclotronData Data Source

The CyclotronData Data Source loads data from a CyclotronData bucket. The bucket must already exist or an error will be displayed. For more details, refer to the documentation on CyclotronData

Please note that it may be necessary to set the optional url property in the Data Source in order to connect to the correct Cyclotron instance. This property lists other Cyclotron servers. By default, this Data Source loads data from the current Cyclotron server.

Pre-Processor

The CyclotronData Data Source supports an optional pre-processor function that will be called before the Data Source executes. For more details, see the main Data Sources page.

Post-Processor

The CyclotronData Data Source supports an optional post-processor function that will be called after data is loaded. For more details, see the main Data Sources page.

Examples

Examples of the CyclotronData Data Source are available on the example-datasource-cyclotrondata dashboard
