Splunk Data Source

The Splunk Data Source runs Splunk jobs synchronously via its API and returns the results. This Data Source is compatible with the Splunk 6 API, and may not work with old versions.

Examples of the Splunk Data Source are available on the example-datasource-splunk dashboard

Pre-Processor

The Splunk Data Source supports an optional pre-processor function that will be called before the Data Source executes. For more details, see the main Data Sources page.

Post-Processor

The Splunk Data Source supports an optional post-processor function that will be called after data is loaded. For more details, see the main Data Sources page.
