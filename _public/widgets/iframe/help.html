iframe Widget

The iframe Widget embeds a web site inside a Cyclotron dashboard. It uses an iframe tag (hence the name), so the target website is loaded completely within the confines of the widget.

Please note that currently, Cyclotron does not have the ability to provide authentication to the website. Pages requiring a login (e.g. Jira) can be used, but they will only work correctly if the browser viewing the dashboard has logged into the website. Otherwise, the widget will most likely show a login screen.

By default, iframe widgets will have scroll bars if their content is too large for the widget size. This can be suppressed by setting the noscroll property to true.

Properties

These are the properties specific to this widget. General Cyclotron widget properties are not repeated here (e.g. layout properties).

Examples

Basic Usage

This is the minimum set of properties needed for this widget.

"widgets": [{
    "url": "http://www.expedia.com",
    "widget": "iframe"
}]

Fullscreen iframe

This dashboard has a single row/column, which is filled with the iframe. The page margin and widget padding is removed with style: "fullscreen", which gives the illusion that the target site is loaded directly. 

{
    "duration": 20,
    "name": "example-fullscreen-iframe",
    "pages": [{
            "layout": {
                "gridColumns": 1,
                "gridRows": 1
            },
            "widgets": [{
                    "gridHeight": 1,
                    "gridWidth": 1,
                    "noscroll": true,
                    "url": "http://www.expedia.com",
                    "widget": "iframe"
                }
            ]
        }
    ],
    "style": "fullscreen",
    "theme": "dark"
}
