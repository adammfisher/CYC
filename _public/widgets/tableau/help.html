Tableau Widget

The Tableau Widget embeds a Tableau view into a Cyclotron widget. This is a simple wrapper for the functionality described in this page.

Properties

These are the properties specific to this widget. General Cyclotron widget properties are not repeated here (e.g. layout properties).
