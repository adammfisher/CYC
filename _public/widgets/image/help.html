Image Widget

The Image Widget is used to display one or more images in a Dashboard Widget. If multiple images are configured and a duration is provided, the Widget will automatically rotate through all the images repeatedly.

Images can be configured with a tooltip that appears on hover, or a link that is opened when the image is clicked.

Size and Position

The default behavior of the Image Widget is to center the image and scale to cover the entire Widget, cropping the sides as needed. This works fairly well for many images and adjusts for various aspect ratios without distorting the image. It may be necessary to modify the defaults for certain images or aspect ratios. The properties follow standard CSS3 properties, so more information can be found on background-image, background-position, background-size, and background-repeat. All of these properties have many possible configurations, many of which cannot be entered via Cyclotron's Editor UI. However, editing the underlying JSON allows one to specify more complex values.

For sizing, the default value is "cover", which scales and covers the entire Widget, cropping the sides. Alternately, "contain" will scale the image to fit the entire image in the Widget, without cropping.

Positioning may be relevent when the image is automatically cropped. Setting a value of "top" or "bottom" will cause the crop to focus on that part of the image, which is useful if the focal point of the image is not in the center.

Basic Usage

At its most basic, this Widget displays an image:

"widgets": [{
    images": [{
        "url": "https://unsplash.com/photos/7BjmDICVloE/download"
    }],
    "widget": "image"
}]

Sizing and Positioning

See above for more information on the size and position features of this Widget.

{
    "images": [{
        "backgroundPosition": "top",
        "backgroundSize": "contain",
        "url": "https://unsplash.com/photos/GYumuBnTqKc/download"
    }],
    "widget": "image"
}

Filters

Various CSS filters are available for modifying the display of the images, and can be combined together. See here for more information, including browser compatibility.

"widgets": [{
    images": [{
        "filters": "contrast(150%) brightness(92%) saturate(150%)",
        "url": "https://unsplash.com/photos/7BjmDICVloE/download"
    }],
    "widget": "image"
}]
