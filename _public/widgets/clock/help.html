Clock Widget

The Clock Widget simply shows the current date and time, which is updated every second.

Properties

This widget has 'Format' text box. Enter the format in which you want to see the time. e.g: 'dddd' in the 'Format' text box gives you only day(Monday or Tuesday etc).

For more formats, referthis

Examples

Basic usage

{
    "widget": "clock"
}

With only Day entered

{
    "format":"ddd",
    "widget": "clock"
}

With nothing entered

{
    "format":"",
    "widget": "clock"
}
