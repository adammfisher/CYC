QRCode Widget

The QR Code Widget generates a QR code out of the current Dashboard URL or some arbitrary text content. When using text content, the text can either be hardcoded or loaded from the first row of a Data Source.

When using a Data Source, only the first row of the result set will be used. Ideally, the Data Source would only return one row, but the filters/sortBy properties can be used to manipulate the result, removing unwanted rows and ordering the correct row at the front (ascending). For example, if a Data Source returns daily data for the past month, sorting by "-date" would order by date descending, causing the row with the most recent data to appear first.

Properties

These are the properties specific to this widget. General Cyclotron widget properties are not repeated here (e.g. layout properties).

Examples

Hard-coded QR Code

This widget generates a QR code from a hard-coded string.

This example is available on the example-qrcode dashboard

{
    "text": "Hello World",
    "widget": "qrcode"
}

Current Dashboard URL

This widget generates a QR code with the current URL.

This example is available on the example-qrcode dashboard

{
    "useUrl": true,
    "widget": "qrcode"
}

Data Source-based QR Code

This widget uses a dynamic value from a Data Source to generate the QR code. If the Data Source automatically refreshes, the QR code will update automatically.

This example is available on the example-qrcode dashboard

{
    "dataSource": "datasource_0",
    "text": "",
    "widget": "qrcode"
}
